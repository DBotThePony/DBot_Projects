
# SCP Insanity

## Welcome!
This is a Garry's Mod addon that adds various SCP objects

## What is [SCP](http://www.scp-wiki.net/faq)?
SCP (Secure, Contain, Protect) - is a collection of fictional "objects" that have abnormal properties
